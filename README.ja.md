日本語版ウィクショナリー向けボットのプロトタイプ

現時点では、ボットによって適用されるべき一連の文字列置換のみが含まれています。

`reading.py` に実装されている主な変換処理は、`{{ja-noun}}`（およびその派生形）テンプレートの更新を行います。

```diff
==={{noun}}===
-[[Category:{{ja}}_{{noun}}]]
-[[Category:{{ja}}_{{noun}}_サ変動詞]]
-{{jachar|思|慮}}（[[しりょ]]）
+{{ja-noun-suru|[[しりょ]]}}
```

具体的には、`main.py`にはウィキテキストを受け取るrepl(acement)関数が含まれており、そのシグネチャは以下の通りです：

```py
def repl(s: str) -> str:
    pass
```

同様に、他の修正のための置換関数もいくつかありますが、それらは優先度が高くありません。
